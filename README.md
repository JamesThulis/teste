# Projeto desenvolvido na plataforma Alura com foco na utilização do Git e do GitHub para controle de versão e colaboração em projetos de software. O objetivo é aplicar na prática os principais conceitos de versionamento, como criação de repositórios, commits, branches, merges e resolução de conflitos, simulando um fluxo de trabalho próximo ao adotado em ambientes profissionais.

Como base para o versionamento, o projeto consiste em uma aplicação web desenvolvida utilizando HTML, CSS e JavaScript, no formato de um jogo para adivinhar um número secreto entre 1 e 100. O HTML é responsável pela estrutura da página, o CSS pela estilização da interface e o JavaScript pela lógica do jogo e pelas interações com o usuário, permitindo compreender de forma prática a integração do Git e do GitHub no desenvolvimento front end e na evolução contínua do código.
